# Public Assets

## `/fonts`
Add font files here (.woff, .woff2, .ttf, .otf)

Example:
```
fonts/
├── GTAmerica-Regular.woff2
├── GTAmerica-Bold.woff2
└── GTAmerica-Extended.woff2
```

## `/illustrations`
All images, icons, and illustrations go here.

Current assets include:
- **Payment cards**: Amex, Apple Pay, Klarna, Depop Balance, etc.
- **Product images**: NB_cap, Cap, Marni_bag, NB_shoes, etc.
- **Icons**: Chevron, Help, Home, Inbox, etc.
- **Illustrations**: Shopping, Empty-Bag, checkout, delivery, etc.
