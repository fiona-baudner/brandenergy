# Brand Energy Prototypes

A collection of interactive UI prototypes showcasing modern web design patterns.

## Prototypes

### 1. Checkout
A complete e-commerce checkout flow featuring:
- Contact information form
- Shipping address input
- Multiple payment method options (Credit Card, Wallet, Bank Transfer)
- Order summary with cart items
- Promo code field
- Order confirmation modal with success animation

### 2. Likes
Interactive social engagement features including:
- Heart animation on click with particle effects
- Double-tap to like (Instagram-style) with floating heart
- Reaction picker with emoji options (Facebook-style)
- Like counters with real-time updates

## Getting Started

Simply open `index.html` in your browser:

```bash
open index.html
```

No build process or dependencies required - these are pure HTML, CSS, and JavaScript prototypes.
